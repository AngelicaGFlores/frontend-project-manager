# About this project.

This project is about creating a full stack Mern App with react, typscript and tailwinds. 

GOAL: This is an app user can log in that will allow them to create a project with a name and description then edit or delete their projects. Once the user creates their project they can create tasks and the then make edits or delete it. 

It has both parts front end and back end.

FRONT END: 
- includes git ignores for .env files and nodes to not be uploaded into github repo
- has pages set up for the project component and possibly tasks component
- has a hook that was not being used
- has a context folder that inlcude authprovider


BACKEND:
- includes the project-task manger repo
- this is the backend to help authenticate users either hard coded or with out it 
- there are schemas and models connected to routes that help make it run 
- server.js includes the middleware
